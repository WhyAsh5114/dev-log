# WhyAsh5114's dev-log

## About
My personal developer portfolio website.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/WhyAsh5114/dev-log
   ```

2. Navigate to the project directory:
   ```bash
   cd dev-log
   ```

3. Install dependencies:
   ```bash
   npm ci --force
   ```

4. Run the project:
   ```bash
   npm run dev
   ```
